# locacao-eventos
Projeto simples simulando locação de propriedades para possíveis eventos.

Obs.: Front-end não é o foco deste projeto.

## Tecnologias utilizadas
- Java EE;
- JSF;
- PostgreSQL;
- Hibernate;
- CDI;
- Lombok;
- -Log4j.
